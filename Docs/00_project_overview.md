## Vision
PupTime is an AI-powered productivity and social scheduling mobile application.

The system helps users:
- Manage personal and social tasks
- Track productivity streaks
- Schedule tasks intelligently
- Collaborate with friends
- Receive AI-based recommendations

## Core Identity

PupTime is NOT just:
- A task manager
- A social app
- An AI wrapper

It is a hybrid productivity intelligence system combining:
- Structured scheduling logic
- AI-assisted optimization
- Social accountability

## Tech Stack

- Mobile: React Native
- Backend: Django + Django REST Framework
- Database: PostgreSQL
- AI: Gemini API
- Async Jobs: (Planned) Celery + Redis
